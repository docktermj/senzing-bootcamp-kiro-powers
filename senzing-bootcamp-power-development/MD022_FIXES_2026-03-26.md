# MD022 CommonMark Fixes - March 26, 2026

## Overview

Fixed MD022 CommonMark errors across multiple markdown files. MD022 requires that headings be surrounded by blank lines (one blank line before and after each heading).

## Error Description

**MD022/blanks-around-headings:** Headings should be surrounded by blank lines [Expected: 1; Actual: 0]

This error occurs when:

- A heading immediately follows content without a blank line before it
- A list or other content immediately follows a heading without a blank line after it

## Best Practices

### When Writing Markdown

1. **Always add blank line before heading:**

   ```markdown
   Previous content

   ### New Heading
   ```

2. **Always add blank line after heading:**

   ```markdown
   ### Heading

   Content starts here
   ```

3. **Lists after headings need blank line:**

   ```markdown
   ### Heading

   - List item
   ```

4. **Code blocks after headings need blank line:**

   ```markdown
   ### Heading

   ```bash
   code here
   ```

   ```
