# Java Standards

- Standard Java conventions, 4-space indent
- `camelCase` methods/variables, `PascalCase` classes, `UPPER_CASE` constants
- Javadoc on all public methods and classes
- No wildcard imports, organized by package
- Validate with: `checkstyle`, `spotbugs`, `PMD`
