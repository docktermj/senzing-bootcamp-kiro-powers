# Senzing Agentic Entity Resolution Bootcamp: Executive Summary

## What It Is

The Senzing Agentic Entity Resolution Bootcamp is a hands-on training program that teaches participants how to solve entity resolution problems using AI-assisted, agentic methods. Participants learn to identify duplicate and related records across one or more data sources, all without machine learning models, training data, or manual rule writing. Instead of hand-coding resolution logic, attendees work with an AI agent that guides data mapping, validates decisions, generates integration code, and accelerates every step from first demo to production deployment. All exercises use open-source datasets, so participants can replicate and extend their work after the bootcamp.

## Why It Matters

Entity resolution traditionally required deep domain expertise, months of tuning, and specialized staff. An agentic approach changes that. The AI agent handles data profiling, mapping, code generation, and best-practice enforcement, letting teams focus on the business problem rather than the underlying infrastructure. Organizations routinely maintain the same customers, vendors, or entities across multiple systems (CRM, billing, support, compliance) with no reliable way to connect them. Entity resolution discovers which records refer to the same real-world entity, enabling a single unified view. The bootcamp gives teams the skills to evaluate, build, and deploy this capability using agentic methods.

## How It Works

Attendees bring a Windows or Mac laptop with a modern web browser and internet connectivity, then connect to a pre-configured AWS bootcamp account that provides the compute environment and sample data. The development workflow pairs Kiro, an AI-assisted IDE, with an MCP-connected entity resolution agent powered by the Senzing SDK. The agent provides context-aware guidance, validated data mappings, and SDK code generation throughout the modules, so participants experience the agentic workflow from the very first exercise. By the end, each participant has a working, deployable entity resolution pipeline built on their bootcamp environment.

## What Participants Will Accomplish

- **See results in minutes.** Module 0 delivers a working entity resolution demo in under 15 minutes using sample data from Las Vegas, London, or Moscow datasets, all driven by the AI agent.
- **Define and collect.** Modules 1 through 3 walk through defining the business problem, collecting source data, and scoring data quality with agent-assisted analysis.
- **Map and load.** Modules 4 through 6 cover mapping fields to the entity specification, setting up the SDK, and loading a first data source, with the agent validating each step.
- **Scale to multiple sources.** Modules 7 and 8 cover multi-source orchestration, query development, and user acceptance testing.
- **Harden for production.** Modules 9 through 12 address performance testing, security hardening, monitoring/observability, and deployment packaging, including Docker, CI/CD, and Kubernetes options.

## Who Should Attend

Data engineers, integration architects, and technical leads who need to solve entity resolution, master data management, or data quality problems. Technical managers evaluating entity resolution solutions will also benefit from the hands-on exposure to agentic methods. The only prerequisites are basic programming in at least one language (Python, Java, C#, or Rust), command-line familiarity, and comfort with CSV and JSON formats. No prior entity resolution, ML, or DBA experience is needed.
