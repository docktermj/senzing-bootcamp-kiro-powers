# Product Context

This repository is the **senzing-bootcamp** Kiro Power — a guided 11-module bootcamp for learning Senzing entity resolution, from first demo to production deployment.

**Users**: Developers learning Senzing entity resolution via Kiro IDE.
**Delivery**: Installed as a Kiro Power; connects to the Senzing MCP server for interactive tool-assisted workflows.
**Key constraint**: Everything in the repo is part of the distributed power. No dev-only files.

The bootcamp produces working code artifacts (transformation, loading, query programs) in the user's chosen language (Python, Java, C#, Rust, TypeScript). The Senzing MCP server generates all SDK code dynamically — no static code templates are shipped.
